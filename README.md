## üíæ Installation

You need to add this line to your Eask file (global recommended).

```elisp
(source 'gnu)
(source 'melpa)
(source 'jcs-elpa)
(source 'eine)
```

Then, install it:

```console
eask install -g x-slime-volleyball
```

## üîß Usage

To run the simple http server:

```console
eask -g exec x-slime-volleyball
```
